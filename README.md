# Bookmark Webpage For Course Platform
> [!NOTE]
> This is a webpage which can store the course platforms.

## How To Use?
> [!TIP]
> #### 0. Install the file above.
> #### 1. Check the file name should be:
> * index.html
> * styles.css
> * script.js
> #### 2. Open the "index.html" by your browser.
